# Meta_Ad_Performance_Dashboard_PowerBI

# 📊 Meta Ad Performance Dashboard

An interactive and visually engaging Power BI dashboard designed to analyze and monitor advertising campaign performance across the Meta ecosystem — including Facebook, Instagram, WhatsApp, and Threads.

---

## 🚀 Project Overview

This dashboard helps marketers and analysts track campaign performance using key advertising KPIs such as:

* Impressions
* Clicks
* Engagements
* CTR (Click Through Rate)
* Conversion Rate
* Purchase Rate
* Budget Analysis
* Audience Insights

The dashboard provides dynamic filtering, trend analysis, demographic breakdowns, and geographical performance tracking to support data-driven marketing decisions.

---

## 🎯 Objectives

* Analyze Meta Ads campaign performance
* Compare engagement metrics across ad types
* Understand audience demographics & interests
* Track campaign trends over time
* Optimize budget allocation and conversion performance

---

## 🛠️ Tools & Technologies

* *Power BI* – Data Visualization & Dashboarding
* *Excel / CSV* – Data Source
* *DAX* – KPI Calculations & Dynamic Measures
* *Power Query* – Data Cleaning & Transformation

## 📌 Dashboard Features

### 🔹 KPI Cards

Displays overall campaign performance metrics including:

* Impressions
* Clicks
* Shares
* Comments
* Purchases
* Engagements
* CTR
* Conversion Rate
* Budget Metrics

### 🔹 Audience Analysis

* Engagement by Gender
* Engagement by Age Group

### 🔹 Trend Analysis

* Weekly Engagement Trend
* Hourly Engagement Trend

### 🔹 Geographic Insights

Interactive map showing engagement distribution by country.

### 🔹 Ad Type Performance

Comparison of:

* Stories
* Carousel
* Video
* Image Ads

### 🔹 Dynamic Filters

Users can filter data by:

* Campaign Name
* Target Interests
* Dynamic Measures
